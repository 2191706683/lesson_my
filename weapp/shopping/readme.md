- 页面开发 新思路
  页面不在由标签构成 ，而由组件构成
  把一个页面任务，分成若干组件开发任务
  分两种组件
  - 一个是可以共享的， 多个页面  就放在components目录下
  - 不共享， 只在特定的页面上出现，页面简洁， 就放到page目录下
