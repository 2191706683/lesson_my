        1. 每个页面都是重复的完整html 结构
        2. 用户体验不太好， 需要优化
            每个页面的请求过程都是一次http的请求响应过程
            完整的html 没有必要
            请求响应需要一个事件  页面会白一下， 页面加载慢
            有什么技术方案 去优化
            SPA  Single Page Application
            单页面开发思维
            ajax   不跳转页面  vue  data
            - SPA
            - ajax  js  动态的更新内容，后台去向服务器端通信，获取新的内容
                不用单纯的借助 url 的切换，http 向服务器重新请求整个页面
